# Zama Skills Roadmap

Our goal is to make `zama-skills` the definitive resource for confidential blockchain development.

## v4.0 (Current)
- [x] Expand to 55+ skills.
- [x] Quality overhaul: Richer documentation and multi-file references.
- [x] Added `EXAMPLES/` folder with full mini-dapps.

## v5.0 (Future)
- [ ] **Interactive Tutorials**: Integration with local devnets for real-time code execution.
- [ ] **Video Series**: HyperFrames-powered walkthroughs for every high-value skill.
- [ ] **Mainnet Readiness**: Complete security checklists for Zama mainnet launch.
- [ ] **Tooling Integration**: CLI command `skills verify` to check contract security.

## Long-term Vision
- Support for FHE-specific Layer 2s and sidechains.
- Cross-chain privacy standards (Encrypted IBC/XCM).
- Hardware-accelerated FHE patterns (HPU).
