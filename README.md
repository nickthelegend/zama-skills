# Zama Skills — Premium FHEVM Collection for AI Agents

Complete, self-contained toolkit for building end-to-end encrypted DApps on Ethereum (Sepolia + mainnet-ready) using Zama FHEVM.

**One-command install:**
```bash
npx skills add nickthelegend/zama-skills
```

### Skills Included (16)
- **Quickstart templates**: Hardhat & React setups.
- **Solidity encrypted types & ops**: Detailed guide on euint types and FHE operations.
- **Fullstack React + Relayer SDK**: Seamless frontend integration.
- **Confidential tokens, NFTs, auctions, games**: Production-ready code for ERC7984, encrypted NFTs, sealed-bid auctions, and Wordle.
- **Testing, deployment, security, troubleshooting**: Comprehensive guides on mock testing, auditing, and debugging.

Perfect for AI coding agents to autonomously build private smart contracts and DApps.
