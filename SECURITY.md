# Security Policy

## Reporting a Vulnerability
We take the security of Zama FHEVM implementations very seriously. If you find a vulnerability in any of our skills or reference implementations, please do not open a public issue. Instead, email security@example.com (placeholder).

## Supported Versions
| Version | Supported |
| :--- | :--- |
| 6.x.x | ✅ Yes |
| 5.x.x | ❌ No |
| < 5.x.x | ❌ No |

## Disclosure Policy
We will acknowledge your report within 48 hours and provide a fix within 7 days for critical vulnerabilities.
